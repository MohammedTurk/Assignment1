# To Do List Application
# Abstract
This repo contains the code for  the frontend that uses React. This is a simple TODO list that has **three pages**.
*Home.
*New ToDo.
*All ToDo.
### And Used Technologies: 
HTML,CSS,Javascript(Reactjs) and Localstorage.
